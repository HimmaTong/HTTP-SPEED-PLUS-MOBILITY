Basic local setup

1. Install Node.js
2. Start server from HTTP\SMServer. Command: "node server.js" 
3. Build client in Visual Studio, or use supplied binary drop.
4. Start client.exe
5. In client app input: CONNECT ws://localhost:8080
6. In client app input: GET files/text.txt


Basic web setup

1. make sure you can connect in your browser to http://smserver.cloudapp.net:8080/
2. Start client.exe
3. In client app input: CONNECT ws://smserver.cloudapp.net:8080
4. In client app input: GET files/text.txt
